# Logical Operations

```java
package proj1;
public class great {
	public static void main(String[] args) {
		int a,b,c;
		a=10;
		b=20;
		c=30;
		System.out.println("a is greatest :"+(b<a&&c<a));
		System.out.println("b is greatest :"+(a<b&&c<b));
		System.out.println("c is greatest :"+(a<b&&b<c));
	}
}
```
